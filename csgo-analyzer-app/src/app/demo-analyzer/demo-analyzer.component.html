Demo Analyzer

This feature is not finished. To use follow these steps:
  1. 1. Install  NodeJs
  2. 2. Install the  demofile  library with this command: npm install --save demofile
  3. 3. Download  DemoParser.js
  4. 4. Run DemoParser with the demofile you want to analyse
       For example:
       node DemoParser.js demo.dem > testdemo.csv
  5. 5. Upload the newly generated file
